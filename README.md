# auto-merge-bot
![](flow-merge.png "Auto Merge Bot")
>  A GitHub App built with [Probot](https://github.com/probot/probot) that Automatically merges pull requests labeled merge-when-ready if all checks pass

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```
